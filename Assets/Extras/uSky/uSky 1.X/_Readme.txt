uSky : Skybox-Based Lighting Shader

REQUIREMENTS
uSky requires Unity 5 Personal or Pro.
Skybox Shaders require Shader Model 3.0 or equivalent.


INSTALLATION 
Import the uSky package file into your project. 
If update from older uSky version. Best way is that remove the older uSky and then import latest version.

LANUCH USKY SCENE
The "uSky Simple Scene" file locates in Scenes folder.

SETUP USKY FROM NEW SCENE:
1) Drag the uSkyManager Prefab to the Scene or Hierarchy window. 
2) Apply Directional Light to uSkyManager as Sun Light.


After the you launch uSky Scene, then you can drag "Timeline" slider to see the day/night sky cycle.
